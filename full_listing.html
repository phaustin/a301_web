sat_lib.modismeta_read.parseMeta(meta_source)[source]¶

Read useful information from a CoreMetata.0 attribute

Parameters

meta_source (str or Path object to file or string with metadata) –

Returns

  • outDict (dict) – key, value:

  • lat_list (np.array) – 4 corner latitudes

  • lon_list (np.array) – 4 corner longitudes

  • max_lat (float) – largest corner latitude

  • min_lat (float) – smallest corner latitude

  • max_lon (float) – largest corner longitude

  • min_lon (float) – smallest corner longitude

  • lon_0 (float) – center longitude

  • lat_0 (float) – center latitude

  • daynight (str) – ‘Day’ or ‘Night’

  • starttime (str) – swath start time in UCT

  • stoptime (str) – swath stop time in UCT

  • startdate (str) – swath start datein UCT

  • orbit (str) – orbit number

  • equatordate (str) – equator crossing date in UCT

  • equatortime (str) – equator crossing time in UCT

  • nasaProductionDate (str) – date file was produced, in UCT

Modis channel properties taken from https://modis.gsfc.nasa.gov/about/specifications.php

chan_dict is a dictionary indexed by channel – i.e. sat_lib.chan_dict[‘1’] gives channel information for channel 1 in EV_250_Aggr1km_RefSB
